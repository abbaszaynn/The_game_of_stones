# **App Name**: MineSight Global

## Core Features:

- Company Profiles: Display detailed profiles for mining companies, including history, leadership, and project details.
- Interactive Map: Integrate Google Maps with polygon overlays to visualize mine locations and boundaries.
- Virtual Mine Tours: Embed Google Earth 3D views or KML/KMZ files for immersive virtual tours of mine sites.
- Document Repository: Store and allow users to search geological reports, licenses, and concession papers as PDFs.  Form data stored in Firestore.
- Real-time News Feed: Display a blog-style feed with company announcements and industry news, automatically posting through social media integration.
- AI-Powered Assistant: Implement an AI tool that can understand and answer user queries based on website data, including reports and news, effectively serving investors.
- Data storage: Leverage Firestore for storing companies, news, documents and inquiry data.

## Style Guidelines:

- Primary color: Deep professional blue (#2E4A62) for text and key elements.
- Background color: Very light, slightly cool grey (#F0F4F7) for a clean backdrop.
- Accent color: Rich vibrant gold (#D4AF37) to highlight calls to action and important information.
- Font: 'Inter' (sans-serif) for all text elements, ensuring clarity and modernity across headings and body.
- Use a sticky header with a semi-transparent background for improved navigation.
